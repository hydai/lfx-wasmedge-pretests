# WasmEdge LFX Mentorship Pre-tests

This repository hosts pre-test requirements for WasmEdge LFX Mentorship projects.

## Usage

**For Mentees:**
Find your project in the [Index](index.html) to view specific pre-test tasks.

**For Mentors:**
1.  **Add File**: Create `YEAR-lfx-term-TERM-ISSUE.html` (copy an existing file as a template).
2.  **Update Index**: Add a link to your project in `index.html`.

## Naming Convention
`YEAR-lfx-term-TERM-ISSUE.html`
Example: `2026-lfx-term-1-4516.html`
